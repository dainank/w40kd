# w40kd
Warhammer 40,000: Darktide performance configs and binaries.

- The two `*.dll` **binary** files should be placed under `*\Steam\steamapps\common\Warhammer 40,000 DARKTIDE\binaries`
    - If you do not trust the `*.dll`'s here, you can download them directly from __Microsoft__ [here](https://devblogs.microsoft.com/directx/directstorage-api-downloads/). 

- The two `*.ini` **config** files should be placed under `*\Steam\steamapps\common\Warhammer 40,000 DARKTIDE\bundle\application_settings`

>[!TIP]
> If you are unsure where your `Warhammer 40,000 DARKTIDE` folder is located, you can find it via Steam through the following steps:
> 1. _Right-click_ on your library entry of the game and select **Properties...**.
> 2. Navigate to the **Installed Files** section.
> 3. Click the **Browse...** button.
> 
> This will open your File Explorer at the `*\Steam\steamapps\common\Warhammer 40,000 DARKTIDE` location, thus you can find the correct locations from there.
